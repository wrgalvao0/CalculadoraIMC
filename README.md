# Calculadora de IMC

## 📋 Descrição

Uma aplicação web simples e intuitiva para calcular o Índice de Massa Corporal (IMC). Permite aos usuários inserir sua altura e peso para obter o cálculo do IMC e a classificação correspondente, ajudando na conscientização sobre saúde e bem-estar.

## 🎯 Objetivo

O objetivo deste projeto é fornecer uma ferramenta acessível e gratuita para que qualquer pessoa possa calcular seu IMC de forma rápida e precisa, promovendo a educação sobre saúde corporal e incentivando hábitos saudáveis.

## ✨ Funcionalidades

- **Entrada de Dados**: Campos para inserir altura (em metros) e peso (em quilogramas).
- **Cálculo Automático**: Calcula o IMC com base na fórmula padrão (peso / altura²).
- **Classificação Visual**: Exibe o resultado com cores indicativas para diferentes faixas de IMC:
  - Magreza (amarelo)
  - Normal (verde)
  - Sobrepeso (laranja)
  - Obesidade (amarelo)
  - Obesidade Grave (vermelho)
- **Limpeza de Campos**: Botão para resetar os inputs e resultados.
- **Responsividade**: Design adaptável para dispositivos móveis.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica da página.
- **CSS3**: Estilização responsiva e moderna.
- **JavaScript (ES6+)**: Lógica de cálculo e manipulação dinâmica do DOM.

## 🚀 Como Executar

### Pré-requisitos
- Um navegador web moderno (Chrome, Firefox, Safari, etc.).

### Passos
1. Clone este repositório:
   ```bash
   git clone https://github.com/wrgalvao0/CalculadoraIMC.git
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd CalculadoraIMC
   ```
3. Abra o arquivo `index.html` diretamente no navegador ou use um servidor local (opcional).
